1모델 체형 분석

업로드한 모델 전신 사진을 AI가 보고 키·둘레·팔다리 길이 같은 신체 실측을 추정합니다. 값이 실제와 다르면 직접 고칠 수 있고, 고친 값이 피팅 지시사항 비교의 기준자로 쓰입니다.

[모델 체형 분석]을 누르면 키/몸무게/어깨너비/가슴·허리·엉덩이둘레/팔길이/상체길이/다리길이가 항목별로 표시됩니다.

2옷 실측 분석

업로드한 상의·하의 사진을 AI가 분석해, 작업지시서(사이즈 실측표)에 나올 법한 실측 수치(cm)를 추정하고, 사진 위에 각 치수의 측정 위치를 화살표로 표시합니다. 값이 실제와 다르면 직접 고칠 수 있고, 고친 값이 다음 단계에 그대로 반영됩니다.

분석 모델

[옷 분석]을 누르면 상의·하의 각각 "무슨 옷인지" 설명과 치수 화살표가 그려진 사진이 위에 나오고, 그 아래에 총장·어깨너비·가슴단면 같은 실측 항목이 하나씩 나뉘어 표시됩니다. 항목마다 직접 고칠 수 있습니다.

3핏 / 기장 지시사항

위 옷 실측과 모델 실측(measurement-model.txt)을 비교해, "소매가 손등 절반을 덮는다", "밑단이 무릎 아래로 내려온다"처럼 신체 기준의 피팅 지시사항을 만듭니다. 여기서 문장을 직접 수정/추가하면 그 조정이 최종 이미지에 그대로 반영됩니다(참조이미지 방식과 달리 텍스트라 무시될 여지가 없습니다).

[피팅 지시사항 생성]을 누르면 상의 / 하의 / 전체로 나뉘고, 핏·기장·소매·허리 같은 항목별로 정리되어 여기 표시됩니다. 항목마다 직접 편집할 수 있습니다.

4피팅샷 생성

모델·상의·하의 원본 3장(원피스 모드는 2장)과 위 피팅 지시사항을 함께 보내 최종 피팅샷을 합성합니다(prompt-23.txt). 여러 번 눌러 여러 장 생성할 수 있습니다.

생성 모델 (복수 선택)

사용법 (v2.3 — 참조이미지 없이 텍스트 지시로 핏/기장 조절)

v2.2의 "더미 참조이미지" 방식은 이미지로 전달한 기장/핏 조정을 AI가 자주 무시하는 문제가 있었습니다.

v2.3은 대신 ① 옷 사진에서 실측(cm)을 추정하고 ② 모델 실측과 비교해 신체 기준 문장으로 된 피팅 지시사항을 뽑은 뒤 ③ 그 텍스트를 최종 프롬프트에 직접 넣습니다. 각 단계 결과를 화면에서 보고 직접 편집할 수 있어, 원하는 핏/기장을 문장으로 확실히 지시할 수 있습니다.
