1모델 체형 분석

모델 전신 사진에서 키·둘레·팔다리 길이를 추정합니다. 이 값이 신체 세로 사다리(어깨·허리·골반·무릎·발목 높이)의 기준이 됩니다. 직접 수정 가능합니다.

[모델 체형 분석]을 누르면 신체 실측이 항목별로 표시됩니다.

2옷 실측 분석 — 레이어별 · 비율 잠금

업로드한 옷을 겹마다 따로 분석합니다(상의 L1 이너 → L2 겉 …). 사진의 절대 cm는 알 수 없지만 비율은 정확하므로, 🔒 잠긴 항목끼리는 픽셀 비율로 연동돼 아는 실측 하나만 고치면 나머지가 함께 재계산됩니다. 특정 값을 빼려면 자물쇠를 🔓 푸세요. 액세서리·신발은 실측 없이 이름만 인식합니다.

분석 모델
🔒 잠김 = 비율 연동 · 🔓 풀림 = 독립 · "AI 추정" 태그 = 아직 AI가 추측한 초기값(고치거나 연동되면 사라짐)
옷 (겹별)
액세서리
신발

[옷 분석]을 누르면 옷의 각 겹, 액세서리, 신발이 각각의 영역에 같은 형식으로 표시됩니다.

3핏 / 기장 지시사항

코드가 겹마다 착지점을 계산하고 겹침 관계(어느 밑단이 더 아래라 무엇이 가려지는지)까지 판정합니다. 선택한 모델은 그 결과를 문장으로 옮기기만 하므로 산수 오류가 없습니다.

핏 모델

[피팅 지시사항 생성]을 누르면 옷별로 정리된, 편집 가능한 지시사항이 표시됩니다.

4피팅샷 생성

모델 + 모든 옷 레이어 + 액세서리 + 신발을 함께 보내 합성합니다(prompt-25.txt). 이미지 역할표는 업로드 구성에 맞춰 자동 생성되며, 업로드하지 않은 슬롯은 아예 지시에 등장하지 않습니다(없는 걸 지어내지 않음).

생성 모델 (복수 선택)

사용법 (v2.5 — 레이어링 + 액세서리 + 신발)

v2.4의 "코드가 착지점을 계산하고 LLM은 문장화만" 구조를 그대로 유지하면서, 한 벌씩만 가능하던 구성을 겹쳐 입기로 확장했습니다.

상의 최대 3겹(셔츠→가디건→코트), 하의 2겹, 원피스/세트 3겹까지 업로드할 수 있고 업로드 순서가 곧 착장 순서(첫 장이 가장 안쪽)입니다. 코드가 겹마다 밑단 착지점을 계산한 뒤 어느 옷이 어느 옷을 덮는지까지 판정해 지시사항에 넣습니다. 액세서리(2)·신발(1)은 실측 없이 이름만 인식해 착장하며, 업로드하지 않으면 착용하지 않습니다(신발만 미업로드 시 AI가 자동 매칭).
